# 发布工作流
在项目正常开发，预览效果达到要求的情况下，可以构建项目来进行更多的测试。目前 Cocos3D 仅支持 Web 和 微信两个平台的构建，更多的构建平台，会在后续的版本中持续推进。目前构建的大体参数流程等，与 Cocos Creator 是基本一致的，但在构建面板的使用上以及生成包体的位置会有一些不同，建议不管是新手还是从 Creator 迁移过来的用户都阅读一遍 `熟悉构建发布面板` 的章节。
